# ⚡ Energy Dashboard Pro

Dashboard interattiva per monitorare e analizzare dati energetici da impianto fotovoltaico.

## 🚀 Features

- 📂 **Upload fino a 10 file** Excel/CSV contemporaneamente
- 📈 **Grafici interattivi** con Plotly (zoom, pan, hover, export)
- 📊 **KPI in tempo reale**: produzione, consumo, autoconsumo, autonomia
- 🔀 **Diagramma Sankey** del flusso energetico
- 🗓️ **Heatmap calendario** della produzione
- 🏆 **Comparazione multi-file** (es. anno vs anno)
- 📥 **Export Excel** dei dati elaborati
- 🎨 **Dark theme professionale**
